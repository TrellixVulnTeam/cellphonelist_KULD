# Cellphone Directory Mini App
Cellphone Directory App with the FEAN Stack (Firebase, Express, Angular and Node).  Deployment with OpenShift.

Preview it at: 

http://cellphone-directory-cellphonelist.193b.starter-ca-central-1.openshiftapps.com/ <br>
<br>user: demo@demo.com<br>pass: demo0606


Features/Technologies:

<b>Firebase</b><br>
 - Database<br>
 - Storage<br>
 - Auth Passport<br>

<b>Express</b><br>
 - API routing for CRUD operations<br>

<b>Angular</b><br>
 - Angular UI routes (for views)<br>
 - Authentication Service (with NodeJs as a middleware between client and firebase)<br>
 - Pagination<br>
 
<b>Node</b><br>
- Holds it all together ;)
